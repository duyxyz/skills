
## Skills hiện có

| Skill | Mô tả |
|---|---|
| `strict-protocol` | Tuân thủ nghiêm ngặt mệnh lệnh, giao tiếp súc tích và phản hồi bằng tiếng Việt |
| `quality-control` | Truy vết lỗi, ghi log có cấu trúc, lập trình phòng vệ và checklist debug |
| `clean-code` | Phân tách logic 3 lớp, quy chuẩn đặt tên và kiểm thử bắt buộc |
| `flutter-setup` | Thiết lập dự án Flutter Android-only theo hướng Material 3, clean architecture theo feature và quy trình build APK |
