## Skills Hiện Có

| Skill | Mô tả |
|---|---|
| `strict-protocol` | Tuân thủ nghiêm ngặt mệnh lệnh, giao tiếp súc tích và commit tiếng Việt |
| `quality-control` | Truy vết lỗi, ghi log có cấu trúc, lập trình phòng vệ và checklist debug |
| `clean-code` | Phân tách logic 3 lớp, quy chuẩn đặt tên và kiểm thử bắt buộc  |
| `flutter` | Cấu hình đơn giản để làm app với flutter  |
